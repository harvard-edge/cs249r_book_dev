~55 minCapstone

The Design Ledger

Every lab asks you to make predictions and design decisions. The Design Ledger saves your choices in your browser and carries them across all 33 labs — from Lab 00 through the final capstone. Lab 08's training memory budget builds on Lab 05's activation analysis, which builds on Lab 01's magnitude calibration.

Your ledger is personal. It lives in your browser's localStorage. If you switch devices, you can export and import it.

Advanced: Run Locally

For the full experience with MLSys·IM's physics engine, run labs locally:

pip install mlsysim[labs]
git clone https://github.com/harvard-edge/cs249r_book.git
cd cs249r_book/labs
marimo run vol1/lab_01_ml_intro.py
